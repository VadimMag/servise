git add . - пометить всё
status - посмотреть
commit -m "текст об изменениях" - подготовка к отправке на сервер
push - отправить на сервер
pull - скачать с сервера последнюю залитую версию
//fetch --all
reset --hard origin/master - тупо загрузить с сервака последнюю версию

branch - просмотр веток
branch asd - создание ветки asd
checkout asd - переход на ветку asd
merge asd - если находимся в мастере то все комиты сливаются в мастер с asd после push и сервак сливает

branch -d asd - удаление ветки локально
push origin --delete asd - удаление на серваке
